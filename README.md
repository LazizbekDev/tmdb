# 🎬 TMDB Telegram Bot - Your Ultimate Movie Companion

Welcome to the **TMDB Bot**! This is your personal hub for finding, watching, and tracking your favorite movies and TV series directly on Telegram without leaving the app.

---

## ✨ What Does This Bot Do?

We believe that discovering what to watch next should be fast and simple. With this bot, you can instantly search for content, view rich cinematic details (trailers, genres, durations), safely store them to your personal Watchlist, and get weekly personalized suggestions!

### 🌟 Key Features
- **Instant Search:** Find any movie or series anywhere using Telegram's Inline Search feature (just type `@bot_username <Movie Name>`).
- **Direct Play:** Receive private video files and episodes straight from the bot to watch securely.
- **Save for Later:** Spotted an interesting movie but busy? Tap **"📌 Add to Watch List"** to save it and type `/watchlist` to view them whenever you're free.
- **Smart Weekly Recommendations:** Not sure what to watch? The bot curates automated weekly suggestions just for you to keep the popcorn flowing!
- **Submit Feedback & Requests:** In the mood for a film we haven't uploaded yet? Submit a direct request to our admins straight from the bot interface.

## 🚀 How to Use the Bot?

You can interact with the bot through simple commands and intuitive buttons. Here are the primary commands you need to know:

| Command | Description |
| :--- | :--- |
| `/start` | Launches the bot and displays the interactive main menu. |
| `/list` | Browse our entire collection of Movies and Series using a clean, paginated menu! |
| `/watchlist` | Quickly view all movies you've personally saved for later. |
| `/suggest` | Receive an instant curated movie recommendation to watch today. |

### 🔍 Inline Search Action
You don't even have to be in the bot's chat to find a movie! 
1. Open any chat with a friend or group.
2. Type our bot's handle, for example: `@your_bot_username Avengers`.
3. A popup menu will appear with the results. Click it to share the trailer and watch link instantly!
